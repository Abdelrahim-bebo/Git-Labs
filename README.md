# Git History & Undo Lab 🛠️

## Objective 🎯
This lab focuses on mastering Git's powerful history manipulation tools, specifically learning how to manage commits and fix mistakes.

## Tasks ✅
1. **Initialize & Populate**: Create a 10-line Python script and make 4 distinct commits.
2. **Reset**: Use `git reset --hard` to remove the last 2 commits.
3. **Continue History**: Add a new commit with the message "commit after deletion".
4. **Amend**: Update the file and use `git commit --amend` to modify the last commit with the message "finish".
5. **Revert**: Practice using `git revert` to undo changes safely.

## Key Commands 💻
* `git init` - Start the repo.
* `git reset --hard HEAD~n` - Delete the last *n* commits.
* `git commit --amend` - Update the most recent commit.
* `git revert` - Create a new commit that undoes a previous one.
